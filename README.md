## 📌 Project Overview

EduPro AI Forecasting & Recommendation Platform is an end-to-end Machine Learning solution designed to help online learning platforms forecast future course demand and predict revenue trends using historical educational data.

The platform combines predictive analytics, business intelligence, and interactive visualizations to support strategic decision-making for educational organizations.

### Business Problems Solved

✅ Predict future course demand

✅ Forecast expected revenue

✅ Identify high-performing course categories

✅ Support instructor hiring decisions

✅ Improve pricing strategies

✅ Enable data-driven business growth

# 🤖 Machine Learning Models

The project evaluates multiple regression models.

| Model                   | R² Score |
| ----------------------- | -------- |
| Linear Regression       | 0.79     |
| Decision Tree Regressor | 0.86     |
| Random Forest Regressor | 0.91     |

### 🥇 Final Selected Model

Random Forest Regressor

Reason:

* Highest Prediction Accuracy
* Better Generalization
* Reduced Overfitting
* Strong Revenue Forecasting Performance
