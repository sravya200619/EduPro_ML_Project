# 🎓 Predictive Modeling for Course Demand and Revenue Forecasting on EduPro

## 📌 Project Overview

This project focuses on building an AI-powered forecasting system for EduPro, an online learning platform.  
The system predicts:

- 📈 Course Enrollment Demand
- 💰 Revenue Forecasting
- 📊 Category-wise Revenue Analysis
- 🧠 Feature Importance Insights

The goal is to help EduPro make data-driven decisions for:

- Launching new courses
- Optimizing course pricing
- Instructor onboarding
- Business growth planning

# 🛠️ Installation & Setup

## Step 1 — Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/EduPro_ML_Project.git
```

---

## Step 2 — Open Project

```bash
cd EduPro_ML_Project
```

---

## Step 3 — Create Virtual Environment

### Mac/Linux

```bash
python3 -m venv venv
```

Activate:

```bash
source venv/bin/activate
```

---

## Step 4 — Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

## Step 1 — Preprocess Data

```bash
python3 preprocessor.py
```

---

## Step 2 — Train Model

```bash
python3 train_model.py
```

---

## Step 3 — Run Streamlit App

```bash
cd app
streamlit run app.py
```
